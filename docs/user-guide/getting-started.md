# Getting Started

## Installation

Download the latest release from [GitHub Releases](https://github.com/phillip-Nexus-Foundry/Censor-Bars/releases).

### Windows
Run the `.msi` installer or extract the portable `.exe`.

### macOS
Mount the `.dmg` and drag Censor Bars to Applications.

### Linux
Install via `.deb`, `.rpm`, or `.AppImage`.

## First Launch

1. Launch Censor Bars — the Control Panel opens.
2. Click **New Bar** (or `Ctrl+N`) to create your first censor bar.
3. The bar appears as a dark overlay — drag it over content you want to hide.
4. Resize from any edge or corner.
5. **Right-click** the bar to access the style menu.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New bar |
| `Ctrl+H` | Hide all bars |
| `Ctrl+Shift+H` | Show all bars |

## Tips

- Create multiple bars for different areas of your screen.
- Use relaxing animations (Ocean Wave, Aurora) for pleasant backgrounds during long calls.
- Use solid dark colors for maximum privacy.
- Adjust opacity if you want to dim rather than fully hide content.
